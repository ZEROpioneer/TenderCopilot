# TenderCopilot Web 界面

本地 Web 操作界面，用于查看报告、手动触发运行、编辑配置、查看爬取历史。

## 运行方式

在项目根目录执行：

```bash
# 安装依赖（若未安装）
pip install fastapi "uvicorn[standard]"

# 启动（默认 http://127.0.0.1:8000）
uvicorn web.app:app --reload --host 127.0.0.1 --port 8000
```

浏览器打开 http://127.0.0.1:8000 即可使用。

## 功能

- **报告**：查看最近日报列表，点击可阅读 Markdown 报告全文。
- **立即运行**：点击按钮触发一次完整流程（与 `py main.py --mode once` 等价），运行期间可查看状态。
- **配置**：查看并修改部分配置（如推荐阈值、爬虫参数），保存后写回 `config/*.yaml`。
- **历史**：查看爬取历史记录表格及近 30 天爬取量折线图。

## 说明

- 与命令行、定时任务并存；界面不替代 `main.py`，仅调用与展示。
- 敏感项（API Key、Webhook 等）在界面上仅显示“已配置”，需在 `.env` 中修改。
