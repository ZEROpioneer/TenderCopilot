## 🐛 故障排查

### 问题1: 搜索框未找到

**症状**:
```
⚠️ 未找到搜索框或搜索失败
```

**解决方法**:
1. 手动访问网站，查看搜索框的HTML结构
2. 修改 `src/spider/plap_spider.py` 中的 `_perform_search()` 方法
3. 调整搜索框选择器

### 问题2: 日期解析失败

**症状**:
```
⚠️ 发现旧公告（None），停止爬取
```

**解决方法**:
修改 `src/spider/plap_spider.py` 中的 `_parse_date()` 方法，添加新的日期格式

### 问题3: API调用失败

**症状**:
```
⚠️ AI提取失败: API error
```

**解决方法**:
1. 检查 `.env` 文件中的 API Key
2. 检查网络连接
3. 查看 API 配额是否用完

### 问题4: 附件下载失败

**症状**:
```
❌ 下载失败: xxx
```

**解决方法**:
1. 检查附件URL是否有效
2. 检查网络连接
3. 检查 `data/attachments/` 目录权限
